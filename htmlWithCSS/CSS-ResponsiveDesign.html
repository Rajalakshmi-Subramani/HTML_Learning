Responsive Design Layout

A Responsive Design adjust the layout design as per the screen. It autofits the screen. Good for Desktop and mobile users.

This is achived by a 12-column grid layout.It is 12 column layout because it is easily divisible by 1,2,3,4 and 6.We can have 3 four columns or 2 6 columns as needed. Media Queries are also for help. Media Queries changes the styles of certain elements defined based on the screen size.
