CSS Selectors

CSS has element selectors , id selectors and class selectors Element and class selectors are more generic and id selectors are specific to the id element. It is not preferred to give an "id" for every element. The css rule for the elements will change if there is an "id" element for every element.

CSS Rules

    • CSS Selector-Rule 1: works like select all
    • Syntax : selector1,selector2{}
    • Meaning: applies css for selector 1 and selector 2.
    • Example: p,.higlight(Apply css for paragraph and .highlight)
    • CSS Selector Rule-2:applies a filter like 'in'
    • Syntax : selector1 selector2{}
    • Meaning: Look for Selector1 then look for selector2. If selector2 is found anywhere down as descendant apply the style. Example: p .highlight(In a paragraph,look for " .highlight" class if found then apply the css rule)
    • CSS Selector Rule 3:applies for immediate child
    • Syntax : selector1>selector2{}
    • Meaning: If Selector2 is immediate child of selector 1 then apply the style.
    • Example: div>p(Look for div, if a paragraph element is immediate descendant then apply the css rule)
