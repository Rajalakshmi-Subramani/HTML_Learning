CSS Rules

Rule 1: Simple Rule/Rule of Precedence. The latest precedent/declared rule of css will be applied for the element.If inline style is defined,it is applied. It is not advisable to declare inline styles as inline style are used for testing.

Rule 2: If an external stylesheet is declared overriding/re-defining the previous one ,then those styles are applied.This includes more than one css include in the html file.

Rule 3: More specific declaration applied to the element.You can take a number score on the css rule and decide which one takes the highest score. The highest score rule is applied
