CSS PseudoClass Selectors

What is a PseudoClass Selector?

A Pseudoclass selector is used when something cannot be done using normal selectors or if selectors are required based on the user's activity

What is the syntax?

selector:pseudo-class{}

What are the different types of Pseudo Selectors?

:link,:hover,:visited,:active:nth-child(...)
